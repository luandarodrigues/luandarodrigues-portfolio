# Luanda Rodrigues Portfolio V2

Editorial, clear and animated portfolio built with **Astro + React + Tailwind + GSAP**.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy on Vercel

Create a GitHub repository, upload these files, import the repository into Vercel and use the default Astro build settings.

Routes:
- `/` English
- `/pt/` Portuguese
